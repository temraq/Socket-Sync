# Установка

## Program1

1. Перейдите в директорию `Program1`.
2. Создайте build директорию: `mkdir build && cd build`.
3. Запустите CMake и сборку: `cmake .. && make`.
4. Запустите программу: `./Program1`.
5. Вводите данные.
